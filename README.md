## 第二章 - Git 基本術語介紹
### 內容差異<br>Diff(erence)
顧名思義，內容差異就是「你改了什麼」的意思

### 提交（動詞）<br>Commit(verb)
將你的內容差異(diff)（字面意義上）提交到版本控制系統成為「一個新的版本」的動作我們稱為「提交(commit)」

### 版本提交（名詞）<br>(Version) commit(noun)
前面提到的「一個新的版本」本身一樣我們也可以稱做「一個版本提交(commit)」
